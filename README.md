# BeiDou
Understanding to Analyze BeiDou X Satellite Navigation System and Intelligence
Recommended packages for interacting with BeiDou / GNSS systems
Your project currently has no GNSS-specific dependencies installed, so install these in the activated venv:

Serial communication

pyserial
NMEA sentence parsing

pynmea2
Geodesy / coordinate handling

pyproj
geographiclib
pymap3d
Numeric / data processing

numpy
scipy
pandas
Install command

pip install pyserial pynmea2 pyproj geographiclib pymap3d numpy scipy pandas
pip install georinex pyubx2 pyrtcm
